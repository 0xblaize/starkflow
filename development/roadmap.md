# Roadmap

## Roadmap

### Phase 1 — Testnet MVP

Completed on Starknet Sepolia.

* Full deployment on Starknet Sepolia.
* Core execution engine live.
* Gas sponsorship integrated.

### Phase 2 — Mainnet launch

* Security audits.
* Production deployment.
* Liquidity scaling.
* Launch of advanced prediction markets.

### Phase 3 — Cross-chain gasless infrastructure

StarkFlow expands beyond Starknet while keeping Starknet as the core coordination layer.

* Extend gasless execution to Ethereum, Solana, and other L2s.
* Keep execution logic and automation coordinated on Starknet.
* Standardize gas abstraction and execution flows.
* Release LazorKit for developer integrations.
