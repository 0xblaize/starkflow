# Solution Overview

## Solution overview

StarkFlow turns complex onchain actions into one gasless flow.

Users interact through one interface.

StarkFlow handles transaction abstraction, gas sponsorship, cross-chain execution, and automated strategies.

### What StarkFlow does

* Sponsors eligible transactions.
* Routes actions across chains.
* Bundles swaps, bridges, and automation into one flow.
