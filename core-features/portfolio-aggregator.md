# Portfolio Aggregator

## Portfolio aggregator

StarkFlow gives users a unified view of balances across supported networks.

It surfaces multichain positions in one place.

### What it does

* Tracks balances across Starknet, Ethereum, and Solana.
* Reduces the need to switch wallets and dashboards.
* Gives execution features shared portfolio context.
