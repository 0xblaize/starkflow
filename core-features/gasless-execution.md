# Gasless Execution

## Gasless execution

StarkFlow removes upfront gas friction from supported flows.

Users sign the action. StarkFlow handles sponsored execution.

### How it works

1. The user initiates a swap, bridge, or strategy.
2. The wallet signs the request.
3. AVNU Paymaster sponsors the transaction.
4. The action settles on Starknet.

### Why it matters

* No separate gas top-up for supported flows.
* Lower onboarding friction.
* Better fit for repeated actions and automation.
